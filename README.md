# lingerie store server

## Install

> git clone git@github.com:energizet/lingerie-store-server.git\
> cd lingerie-store-server\
> npm ci\
> npm run init

## Config

> /api/items.json - список товаров\
> /wwwroot/* - статические файлы

## Start

> npm run start

## Apis

> GET /api/items\
> GET /api/item/{id}
